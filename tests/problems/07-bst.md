# Problem 7: Binary Search Tree

Implement a binary search tree in Python with the following operations:

- `insert(value)` — add a value
- `delete(value)` — remove a value
- `search(value) -> bool` — check if a value exists
- `inorder() -> list` — return sorted values
- `min()` / `max()` — return min/max values
- `height() -> int` — return tree height
- `is_balanced() -> bool` — check if tree is balanced (height difference ≤ 1 at every node)

Include tests that verify all operations work correctly, including edge cases (empty tree, single node, delete root, etc).

## Deliverable

Write all code to a `bst/` directory.
